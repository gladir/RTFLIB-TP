# RTFLIB-TP
Bibliothèque écrit en Pascal (Turbo Pascal ou Free Pascal) pour le traitement de document RTF.

<h3>Liste des unités</h3>

<table>
  <tr>
    <th>Nom</th>
    <th>Description</th>
  </tr>
  <tr>
    <td><b>RTFBASE.PAS</b></td>
    <td>Cette unité contient des fonctions de base pour le RTF.</td>
  </tr>  
  <tr>
    <td><b>RTFDEF.PAS</b></td>
    <td>Cette unité contient les définitions du RTF.</td>
  </tr>    
  <tr>
    <td><b>RTFEXP.PAS</b></td>
    <td>Cette unité contient des fonctions d'exportations de RTF vers un autre format.</td>
  </tr>
  <tr>
    <td><b>RTFSAVE.PAS</b></td>
    <td>Cette unité contient des fonctions pour sauvegarder un RTF.</td>
  </tr>  
</table>
